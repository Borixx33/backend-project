## Projet API-REST node JS Gestion de membres

Ce projet est une API-REST permettant de voir un ou tous les membres, d'ajouter un membre, 
de le modifier ou encore de le supprimer.



Pour bien commencer

```node
npm install
```
```node
npm start
```
ou

```node
nodemon app.js
```
